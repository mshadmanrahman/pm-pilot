# /tdd

Start test-driven development for a feature or fix.

## Usage
```
/tdd {feature or behavior to implement}
```

## Behavior
1. Invoke the **tdd-guide** agent from `agents/tdd-guide.md`
2. Write failing test first (RED)
3. Implement minimal code to pass (GREEN)
4. Refactor and verify coverage (IMPROVE)
5. Repeat until feature is complete with 80%+ coverage
